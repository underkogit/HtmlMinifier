
 


## HtmlMinifier

**HtmlMinifier** — это инструмент для минимизации HTML-кода, который уменьшает его размер, удаляя ненужные пробелы, комментарии и другие избыточные элементы. 
Проект написан для платформы ESP32 и ему подобных.
